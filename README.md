# Photo Vision Report

A React-based application for generating and managing photo vision reports.

## Technologies Used

- Vite
- TypeScript
- React
- shadcn-ui
- Tailwind CSS
- Supabase (for backend integration)

## Getting Started

1. Clone the repository:
   ```sh
   git clone <YOUR_GIT_URL>
   cd photo-vision-report-main
   ```

2. Install dependencies:
   ```sh
   npm install
   ```

3. Start the development server:
   ```sh
   npm run dev
   ```

## Features

- Dashboard for report overview
- Academic performance tracking
- Assignment management
- Report downloads
- User authentication
- Notifications and settings
